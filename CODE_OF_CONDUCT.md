# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Project-Specific Guidelines

OpenGIKAI is a **politically neutral** public media tool. All parliamentary speeches are processed with the same algorithm regardless of party affiliation.

The following are **out of scope** for this project and will be closed without discussion:

- Requests to alter how specific politicians, parties, or political positions are presented
- Political advocacy, campaigning, or lobbying through issues, PRs, or discussions
- Debates about the content of parliamentary proceedings (use official channels for that)
- Requests to selectively include or exclude specific speeches or sessions

We welcome technical contributions that improve the platform for **all** users equally.

## Enforcement

Instances of unacceptable behavior may be reported to the project maintainers. All complaints will be reviewed and will result in a response deemed necessary and appropriate to the circumstances.
